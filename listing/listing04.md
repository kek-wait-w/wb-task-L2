Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Пока пишутся данные в канал будут выводится числа с 0 по 9, а потом дедлок,
тк читаем только мы, писать никто не может.

```
